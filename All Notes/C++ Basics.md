Import <iostream> for all types of I/O.
Alternatively we can use <bits/stdc++.h> to include all the standard libraries in C++.

Basic template

```cpp
#include <bits/stdc++.h>
int main () {
	cout << "Hello World..!"
	return 0;
}
```

# Data Types

- float
- double
- char
- string & getline()
